# SuperMemo

A TypeScript implementation of the [SuperMemo 2](https://super-memory.com/english/ol/sm2.htm) (SM2) algorithm for spaced based repetition algorithm.



## Usage

### Install

```bash
npm install @kirklin/supermemo2
```
